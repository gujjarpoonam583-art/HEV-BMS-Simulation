# HEV-BMS-Simulation
Hybrid Electric Vehicle (HEV) Battery Management System (BMS) Simulation project developed using MATLAB/Simulink and Python. The project focuses on State of Charge (SOC) estimation, State of Health (SOH) monitoring, cell balancing, thermal analysis, battery protection mechanisms, and performance evaluation under different load conditions.
# HEV-BMS-Simulation

## Project Overview
This project focuses on the simulation of a Hybrid Electric Vehicle (HEV) Battery Management System (BMS) using MATLAB/Simulink and Python.

## Objectives
- State of Charge (SOC) Estimation
- State of Health (SOH) Monitoring
- Cell Balancing
- Thermal Analysis
- Overcharge and Over-discharge Protection
- Performance Evaluation
