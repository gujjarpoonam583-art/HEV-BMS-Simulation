# Hybrid Electric Vehicle Battery Management System

## Objective
To simulate and analyze a Battery Management System (BMS) for a Hybrid Electric Vehicle (HEV).

## Features
- State of Charge (SOC) Estimation
- State of Health (SOH) Monitoring
- Battery Protection
- Thermal Analysis
- Cell Balancing
- Performance Evaluation

## Tools Used
- MATLAB/Simulink
- Python
- GitHub

## Dataset
EV Energy Management Dataset
